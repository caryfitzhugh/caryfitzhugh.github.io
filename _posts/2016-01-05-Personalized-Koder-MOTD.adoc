You need to set the message in /etc/motd

(To do this log in as debian, so you can sudo).

Also - /etc/ssh/sshd_config: Change the setting PrintLastLog to "no", this will disable the "Last login" message.  ANd change PrintMotd to "yes".  This displays the beautifulness.
